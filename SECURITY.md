# Security Policy

## Overview
This project — **Dibassories** — is a university project created for educational and learning purposes only.  
The main goal of this repository is to demonstrate basic web development concepts using **PHP (Vanilla)**, **MySQL**, and common front-end technologies.

Please note that **security is not the main focus** of this project, and the website should **not be used in any real-world or production environment**.

---

## Reporting a Vulnerability
Since this is a **student learning project**, vulnerabilities or security issues are **expected** and are part of the learning process.  
However, if you wish to point out a bug or provide feedback to help improve the code quality, feel free to open an [issue](../../issues) in the repository.

---

## Disclaimer
This project:
- Is **not designed** with production-level security measures.  
- May contain insecure or unoptimized code.  
- Should **not** be deployed on public servers handling real user data.  
- Is intended **only for educational and demonstration purposes**.

By using or reviewing this code, you acknowledge that you understand the **security limitations** of this project.
